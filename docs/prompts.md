# Prompt Notes

## Roadmap Generator

System prompt goals:
- output strict JSON only
- produce week-by-week learning structure
- adapt difficulty, timeline, and study time
- include practice and project tasks per topic
- avoid external links and hallucinated providers

## Resume Analyzer

System prompt goals:
- score ATS fit for the target role
- identify missing keywords
- suggest section-level improvements
- prioritize measurable impact and relevant skills

## Skill Gap Analyzer

System prompt goals:
- compare roadmap progress, resume content, and target role expectations
- output missing skills, priority ranking, and next learning actions
